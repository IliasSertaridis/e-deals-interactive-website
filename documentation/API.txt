 με ορισμα το query που θελουμε να τρεξει στη βαση επιστρεφετε το αποτελεσμα (εστω $result)
 To $result παρεχει διαφορες λειτουργιες οπως:

 	$result->num_rows 	αριθμος γραμμων του αποτελεσματος

 	$row = $result->fetch_assoc()	επιστρεφει γραμμες του αποτελεσματος. Χρησιμοποιειτε σε while ως εξης: while($row = $result->fetch_assoc()) και επιστρεφει καθε φορα την επομενη γραμμη του αποτελεσματος.
 	$row["name_of_column"] επιστρεφετε το περιεχομενο της γραμμης στη συγκεκριμενη στηλη.


Επειδή θέλουμε να φορτώσουμε δεδομένα στη JavaScript μέσω AJAX σύμφωνα με την εκφώνηση αυτό πρέπει να γίνει μέσω κλήσεων  HTTP GET (ή και POST). Οπότε εφόσον "φεύγουμε" από την PHP δεν μπορούμε να διατηρήσουμε τα MySQLi data types, άρα χρειάζεται να πάμε σε JSON encoding (εναλλακτικά XML, επέλεξα JSON).

Οπότε αρχικά το funtion new_query έχει γίνει function DBquery το οποίο επιστρέφει JSON encoded μεταβλητή και μπορεί να χρησιμοποιηθεί όπου χρειάζεται μέσω require ή import σε PHP αρχεία.

Επίσης δημιούργησα το αρχείο api.php στο οποίο δρομολογούνται τα URLs τύπου (/api/$query) με dynamic GET, και χρησιμοποιώντας το DBquery τυπώνει το JSON αποτέλεσμα

Οπότε αυτό το αποτέλεσμα πλέον μπορεί να φορτωθεί μέσω AJAX στη JavaScript, όπως θα μας χρειαστεί.

Παράδειγμα χρήσης AJAX υπάρχουν (για τώρα) στα test.html, test.js, και προφανώς αυτά λειτουργούν συνδυαστικά με το routes.php, api.php, dbquery.php.
